# Correio Elegante Digital IFPA

Aplicação web responsiva desenvolvida com React, TypeScript, Vite e Supabase para a gincana do IFPA.

## Como rodar

1. Instale as dependências:
   npm install
2. Crie um arquivo .env com as variáveis do Supabase:
   VITE_SUPABASE_URL=...
   VITE_SUPABASE_ANON_KEY=...
3. Inicie o projeto:
   npm run dev

## Funcionalidades

- Página inicial com identidade visual institucional
- Envio de mensagens com identificação ou anonimato
- Contador de caracteres e moderação básica
- Painel administrativo com senha simples
- Exportação PDF de mensagens

## Deploy

A aplicação está pronta para deploy em Vercel.
